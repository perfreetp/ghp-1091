## 1. 产品概述

智慧楼宇住户服务助手是一款面向办公楼员工和入驻企业前台的移动端应用，通过整合楼宇通行、访客管理、会议室预订、报修服务、停车管理等核心功能，提供一站式智慧办公体验，提升楼宇运营效率与用户满意度。

- 目标用户：办公楼员工、入驻企业前台/行政人员
- 核心价值：数字化楼宇服务体验，提升办公效率与便捷度

## 2. 核心功能

### 2.1 用户角色

| 角色 | 说明 | 核心权限 |
|------|------|----------|
| 办公楼员工 | 楼宇内办公的普通职员 | 查看公告、扫码通行、预订会议室、报修、停车缴费等 |
| 企业前台/行政 | 入驻企业的前台或行政人员 | 访客预约管理、批量通行申请、企业会议室管理等 |

### 2.2 功能模块

1. **首页**：楼宇公告轮播、快捷功能入口、室内导航、电梯引导、快递通知、应急疏散
2. **通行**：二维码门禁刷卡、临时通行申请、通行记录查询
3. **访客**：访客预约、到访码生成与分享、访客记录、来访登记
4. **会议室**：会议室搜索筛选、预订改签取消、我的预订、会议室详情
5. **报修**：拍照报修、分类选择、进度跟踪、舒适度反馈（温度/照明）
6. **停车**：车位查找、停车费缴纳、车牌绑定、停车记录
7. **我的**：个人信息、企业信息、满意度评价、消息中心、设置帮助

## 3. 核心流程

### 3.1 二维码门禁通行流程
用户打开通行页 → 系统自动生成动态二维码 → 用户将二维码对准闸机扫码区域 → 闸机识别通过并开门 → 系统记录通行信息

### 3.2 访客预约流程
用户进入访客模块 → 填写访客信息表单 → 提交生成到访码 → 通过短信/微信分享到访码给访客 → 访客凭码登记入场 → 系统记录来访状态

### 3.3 会议室预订流程
用户进入会议室模块 → 筛选条件（日期/人数/设备）→ 浏览可用会议室 → 选择时段 → 填写会议信息 → 确认预订 → 收到预订确认通知

### 3.4 报修处理流程
用户发现问题 → 进入报修模块 → 选择分类 → 拍照上传+文字描述 → 提交报修单 → 物业受理派单 → 维修人员处理 → 用户验收评价

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：科技蓝（#1E40AF）作为主色，体现专业与信任感
- **辅助色**：青色（#0EA5E9）用于强调与交互元素，暖橙（#F97316）用于提醒与告警
- **中性色**：深灰（#1F2937）文字、中灰（#6B7280）辅助文字、浅灰（#F3F4F6）背景
- **按钮风格**：圆角大按钮（12px），主色填充，渐变效果，按下微缩放动效
- **字体**：标题使用Noto Sans SC Bold，正文使用Noto Sans SC Regular，整体字号偏大适配移动端
- **布局风格**：底部Tab导航（5个）+ 顶部内容区，卡片式布局，圆角（16px），柔和阴影
- **图标风格**：Lucide线性图标，24px标准尺寸，当前选中项使用主色填充

### 4.3 响应式设计
- 移动优先设计，基准宽度375px（iPhone SE）
- 最大宽度限制480px，居中显示适配平板
- 触控目标最小44x44px，符合WCAG标准
- 底部导航栏高度固定64px，安全区适配iPhone X+
- 横竖屏切换时保持布局稳定性

### 4.4 动效与交互
- 页面切换：左右滑入滑出过渡（200ms ease-out）
- 按钮点击：微缩放（0.97倍）+ 透明度变化
- 列表加载：骨架屏占位 + 渐入效果
- 二维码区域：边框呼吸灯脉动动画（3s循环）
- 下拉刷新：标准Material风格指示器
- 通知角标：出现时弹性缩放动画
